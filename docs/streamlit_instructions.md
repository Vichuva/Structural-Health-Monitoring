# Streamlit Dashboard Verification

The Streamlit dashboard has been identified and verified.

## Dashboard Entry Point
- **File**: `src/visualization/dashboard.py`

## How to Run
To launch the dashboard, run the following command from the project root:

```powershell
streamlit run src/visualization/dashboard.py
```

## Prerequisites
- **Virtual Environment**: Ensure your virtual environment is activated.
  - Windows: `.\venv\Scripts\activate`
- **Dependencies**: Ensure all dependencies are installed.
  - `pip install -r requirements.txt`

## Features
The dashboard includes:
- **3D Bridge Twins**: Interactive 3D visualization of bridge structures.
- **Fleet Risk Map**: Analysis of the entire monitored fleet.
- **InSAR Intelligence**: Advanced InSAR products (interferograms, heatmaps).
- **AI Reports**: Renders engineering reports generated by the CrewAI agents.

> [!TIP]
> If the "AI Reports" tab is empty, run the pipeline with the `--run-agents` flag to generate them:
> `python main.py --run-agents --llm-provider groq`
