# SHM CrewAI Reports

Auto-generated engineering reports from multi-agent analysis.

## Generated Reports

Will appear here after `python main.py --run-agents`:

```
shm_report_YYYYMMDD_HHMMSS.md
├── Executive Summary (for managers)
├── Technical Findings (Data Analyst)
├── Risk Assessment (Triage Agent)  
├── Recommendations (action items)
└── Uncertainty Quantification
```
